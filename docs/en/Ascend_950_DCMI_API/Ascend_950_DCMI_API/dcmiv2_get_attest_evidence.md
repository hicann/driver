# dcmiv2\_get\_attest\_evidence<a name="EN-US_TOPIC_0000002565252107"></a>

## Prototype<a name="en-us_topic_0000001206147244_en-us_topic_0000001223414441_en-us_topic_0000001099619850_toc533412077"></a>

**int dcmiv2\_get\_attest\_evidence\(int dev\_id, struct attest\_ctx \*ctx, unsigned char \*evidence, unsigned int \*evidence\_len\)**

## Description<a name="section1468480762"></a>

Obtains the measurement values \(PCR+Quote\) during device startup, Integrity Measurement Architecture \(IMA\) logs, and BIOS startup logs.

typedef enum {
EVIDENCE_TYPE_NONE = 0, // Failure returned.
EVIDENCE_TYPE_QUOTE = 1, // Only PCR+Quote is included.

struct attest_ctx {
unsigned int challenge_len; // Length of the challenge value.
EVIDENCE_TYPE evidence_type; // Type of the evidence to be obtained.

}

## Example<a name="section1888234213618"></a>

```c
...
struct attest_ctx ctx = {0};
unsigned char evidence[65536] = {0}; // Assumed maximum evidence length.
unsigned int evidence_len = sizeof(evidence); // ctx structure initialization.
ctx.challenge_len = 32;
ctx.evidence_type = 3; // Assumed evidence type: 1, 3, 5, or 7.
```
