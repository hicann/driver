# dcmiv2\_get\_device\_chip\_info<a name="EN-US_TOPIC_0000002347924113"></a>

## Prototype<a name="section1977653135210"></a>

**int dcmiv2\_get\_device\_chip\_info\(int dev\_id, struct dcmi\_chip\_info\_v2 \*chip\_info\)**

## Description<a name="section997810532524"></a>

Obtains the  **dcmi\_chip\_info\_v2**  information of an NPU and a CPU, including  **chip\_name**,  **chip\_ver**,  **aicore\_cnt**, and  **npu\_name**.
