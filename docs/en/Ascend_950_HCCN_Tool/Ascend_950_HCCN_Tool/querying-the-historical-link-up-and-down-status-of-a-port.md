## Function<a name="section957615561617"></a>

Queries a port's historical link up/down status.

## Precautions<a name="section8415102071"></a>

You can query up to 10 historical link up/down status records prior to the current time. The query results are displayed in reverse chronological order.

## Syntax<a name="section4655161312717"></a>

```bash
hccn_tool -g -link_stat -i <dev_id> -u <udie_id> -p <port_id>
```
