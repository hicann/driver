## Function<a name="en-us_topic_0184799386_section1383319120101"></a>

## Syntax<a name="en-us_topic_0184799386_section2083668161015"></a>


## Example<a name="en-us_topic_0184799386_section1684018331898"></a>

- Example 1:

    ```bash
    hccn_tool -h
    ```

    Query the hccn\_tool help information.

    ```text
    Usage
            hccn_tool -mode -cmd -npu_info [-options]
    mode:
            -g      For -cmd and -npu_info, enter [ hccn_tool -h -g ] for more details.
            -s      For -cmd and -npu_info, enter [ hccn_tool -h -s ] for more details.
            -t      For -cmd and -npu_info, enter [ hccn_tool -h -t ] for more details.
    example:
            hccn_tool -g -link -i 0 -u 1 -p 0       query link status
    ```

- Example 2:

    ```bash
    hccn_tool -h -g
    ```

    Obtain help information for  **hccn\_tool**  query commands.

    ```text
    hccn_tool -g -fec           : check fec info
                 -i <device_id> : device id
                 -u <udie_id>   : udie id
                 -p <port_id>   : port id
    hccn_tool -g -ub_connect    : check ub connect status
                 -i <device_id> : device id
    ...
    ```

- Example 3:

    ```bash
    hccn_tool -h -link
    ```

    Obtain help information for the main command word  **link**.

    ```text
    hccn_tool -g -link                                      : get port link status, UP or DOWN
                 -i <dev_id>                                : device id
                 -u <udie_id>                               : udie id
                 -p <port_id>                               : port id
    hccn_tool -s -link                                      : link up or shut down net device
                 -i <dev_id>                                : device id
                 -d <ndev_name>                             : netdev name
                 -up                                        : link up net device
    hccn_tool -s -link                                      : link up or shut down net device
                 -i <dev_id>                                : device id
                 -d <ndev_name>                             : netdev name
                 -down                                      : shut down net device
    ```
