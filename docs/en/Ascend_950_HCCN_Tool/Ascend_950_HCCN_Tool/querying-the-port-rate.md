## Function<a name="section957615561617"></a>

Queries the port link speed.

## Precautions<a name="section8415102071"></a>

None.

## Syntax<a name="section4655161312717"></a>

```bash
hccn_tool -g -speed -i <dev_id> -u <udie_id> -p <port_id>
```

## Example<a name="section185511229978"></a>

```bash
hccn_tool -g -speed -i 0 -u 0 -p 4
```

Query the link speed of port 4 on UDie 0 under logical device ID 0.

```text
+-----+------+------+---------+-----+-------+-----------+
| NPU | UDie | Port | Speed   | Cap | Width | Status    |
+-----+------+------+---------+-----+-------+-----------+
| 0   | 0    | 4    | 400 G   | X4  | X4    | normal    |
+-----+------+------+---------+-----+-------+-----------+
```
