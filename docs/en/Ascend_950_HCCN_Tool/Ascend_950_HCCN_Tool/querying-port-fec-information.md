## Function<a name="section957615561617"></a>

Queries forward error correction \(FEC\) information. FEC is a coding technology that does not require retransmission requests from the receive end. It adds redundant check information at the transmit end so that the receive end can detect and correct transmission bit errors. The core of FEC is to use bandwidth redundancy to ensure transmission reliability and low latency.

## Precautions<a name="section8415102071"></a>

None.

## Syntax<a name="section4655161312717"></a>

```bash
hccn_tool -g -fec -i <dev_id> -u <udie_id> -p <port_id>
```
