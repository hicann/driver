# Querying the Device Power<a name="EN-US_TOPIC_0000002484466734"></a>

## Function<a name="en-us_topic_0000001220096538_en-us_topic_0000001163757099_section18674262006"></a>

This command is used to query the power of a device.

## Syntax<a name="en-us_topic_0000001220096538_en-us_topic_0000001163757099_section386902615016"></a>

`npu-smi info -t power -i dev_id`

## Example<a name="en-us_topic_0000001220096538_en-us_topic_0000001163757099_section66051714132117"></a>

Query the power of device 0.

```bash
npu-smi info -t power -i 0
```

```text
        NPU ID                         : 0
        NPU Real-time Power(W)         : 296.5
```
