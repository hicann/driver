# Querying the SIO Status Between Dies of a Device<a name="EN-US_TOPIC_0000002484306734"></a>

## Function<a name="section17599103415502"></a>

This command is used to query the SIO status between dies of a device.

## Syntax<a name="section1209113910506"></a>

`npu-smi info -t sio-info -i dev_id`

## Example<a name="section1769295716508"></a>

Query the SIO status between dies of device 0.

```bash
npu-smi info -t sio-info -i 0
```

```text
TX Error Count : 0
RX Error Count : 0
```
