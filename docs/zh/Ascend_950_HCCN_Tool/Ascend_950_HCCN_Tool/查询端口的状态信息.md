## 命令格式<a name="zh-cn_topic_0184799386_section2083668161015"></a>

```bash
hccn_tool -g -port_info -i <dev_id> -u <udie_id> -p <port_id>
```

    ```text

    port enable: on
    port link: link
    driver_type: TYPE_NONE
    port_mode: PORT_MODE_SHARED
    ```
