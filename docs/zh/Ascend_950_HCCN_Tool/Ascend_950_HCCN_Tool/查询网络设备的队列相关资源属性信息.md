## 功能说明<a name="zh-cn_topic_0184799386_section1383319120101"></a>

查询队列相关资源属性信息。

## 注意事项<a name="zh-cn_topic_0238914640_section127461744152119"></a>

执行该命令查询队列相关资源属性信息前，可执行[查询网络设备的队列相关资源列表信息](查询网络设备的队列相关资源列表信息.md)命令查看资源列表中是否存在该资源类型，例如回显jfs\_cnt值为0，表示JFS资源类型不存在，不可查询；例如回显jfs\_cnt值不为0，表示JFS资源类型存在。

## 命令格式<a name="zh-cn_topic_0184799386_section2083668161015"></a>

```bash
hccn_tool -g -show_res -i <dev_id> -d <udma_dev_name> -resource_type <type_id> -key_id <key_id>
```

## 使用示例<a name="zh-cn_topic_0184799386_section1684018331898"></a>

```bash
hccn_tool -g -show_res -i 0 -d udma3 -resource_type 6 -key_id 4
```

查询逻辑设备ID为0的udma3设备的ID为4的JFR资源属性信息。

```text
**********RES_JFR**********
jfr_id              : 4
state               : 1 [READY]
depth               : 1024
jfc_id              : 9
```
