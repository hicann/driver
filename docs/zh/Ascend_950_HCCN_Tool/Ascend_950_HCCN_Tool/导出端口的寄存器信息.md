## 功能说明<a name="section957615561617"></a>

导出端口寄存器信息。

## 注意事项<a name="section8415102071"></a>

执行该命令前可参见对应版本《[msnpureport工具使用指南](https://support.huawei.com/enterprise/zh/ascend-computing/ascend-hdk-pid-252764743?category=reference-guides&subcategory=command-reference)》的“设置Device侧系统类日志级别”章节，开启debug日志级别，执行该命令后，可执行**msnpureport -f**搜集NPU侧日志，会在当前路径下生成一个以时间命名的日志文件夹。Device侧日志可在该日志文件夹的/slog/device-os-_X_/debug/device-os或/slog/device-os-_X_/debug/device-_X_路径下查看目标寄存器信息。

## 命令格式<a name="section4655161312717"></a>

```bash
hccn_tool -t -dump -i <dev_id> -u <udie_id> -p <port_id>
```
