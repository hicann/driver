## 功能说明<a name="section957615561617"></a>

查询端口历史建链流程的状态信息。

## 注意事项<a name="section8415102071"></a>

最多可查询5次历史建链信息，回显按时间由近到远呈现。

## 命令格式<a name="section4655161312717"></a>

```bash
hccn_tool -g -adapt_status -i <dev_id> -u <udie_id> -p <port_id>
```

## 使用示例<a name="section185511229978"></a>

```bash
hccn_tool -g -adapt_status -i 0 -u 0 -p 4
```

查询逻辑设备ID为0的UDie 0端口4的历史建链流程的状态信息。

```text
print the adapt status table:
+-----+------+------+-------+-------------------------+-----------------------------------------------------------+
| NPU | UDie | Port | Index | Time_Stamp              | Adapt status: UB                                          |
+-----+------+------+-------+-------------------------+-----------------------------------------------------------+
| 0   | 0    | 4    | 0     | Thu Jan 1 00:00:00 1970 | L0:link_proc|L1:link_fail                                 |
| 0   | 0    | 4    | 1     | Thu Jan 1 00:00:00 1970 | L0:link_proc|L1:wait_detect|L2:linkdown|L3:linkdown_off   |
| 0   | 0    | 4    | 2     | Thu Jan 1 00:00:00 1970 | L0:link_proc|L1:start|L2:switch_speed|L3:switch_speed_off |
| 0   | 0    | 4    | 3     | Thu Jan 1 00:00:00 1970 | L0:switch_speed                                           |
| 0   | 0    | 4    | 4     | Thu Jan 1 00:00:00 1970 | L0:start                                                  |
+-----+------+------+-------+-------------------------+-----------------------------------------------------------+
print the lane raise table:
+-----+------+------+-------+-------------------------+---------------------------------+
| NPU | UDie | Port | Index | Time_Stamp              | Adapt status: LANE raise        |
+-----+------+------+-------+-------------------------+---------------------------------+
| 0   | 0    | 4    | 0     | Thu Jan 1 00:00:00 1970 | L0:start                        |
+-----+-----+------+-------+-------------------------+---------------------------------+
```
