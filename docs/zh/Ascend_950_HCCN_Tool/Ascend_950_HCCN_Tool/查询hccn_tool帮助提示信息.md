## 使用示例<a name="zh-cn_topic_0184799386_section1684018331898"></a>

- 示例1：

    ```bash
    hccn_tool -h
    ```

    查询hccn\_tool工具的帮助提示信息。

    ```text
    Usage
            hccn_tool -mode -cmd -npu_info [-options]
    mode:
            -g      For -cmd and -npu_info, enter [ hccn_tool -h -g ] for more details.
            -s      For -cmd and -npu_info, enter [ hccn_tool -h -s ] for more details.
            -t      For -cmd and -npu_info, enter [ hccn_tool -h -t ] for more details.
    example:
            hccn_tool -g -link -i 0 -u 1 -p 0       query link status
    ```
