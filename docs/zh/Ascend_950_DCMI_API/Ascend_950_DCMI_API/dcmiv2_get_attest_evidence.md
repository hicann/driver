# dcmiv2\_get\_attest\_evidence<a name="ZH-CN_TOPIC_0000002565252107"></a>

## 函数原型<a name="zh-cn_topic_0000001206147244_zh-cn_topic_0000001223414441_zh-cn_topic_0000001099619850_toc533412077"></a>

**int dcmiv2\_get\_attest\_evidence\(int dev\_id, struct attest\_ctx \*ctx, unsigned char \*evidence, unsigned int \*evidence\_len\)**

## 功能说明<a name="section1468480762"></a>

获取设备启动时的度量值（PCR+Quote），以及IMA（Integrity Measurement Architecture）日志和BIOS启动日志。

## 调用示例<a name="section1888234213618"></a>

```c
…   
struct attest_ctx ctx = {0};
unsigned char evidence[65536] = {0}; //假设的最大证据长度
unsigned int evidence_len = sizeof(evidence); //初始化ctx结构体
ctx.challenge_len = 32;
ctx.evidence_type = 3; //假设的证据类型 支持1,3,5,7
dcmiv2_init();//生成随机challenge
srand((unsigned int)time(NULL));
for (int i = 0; i < MAX_CHALLENGE_LEN; i++) {
ctx.challenge[i] = (char)(rand() % 256);
}
int ret = dcmiv2_get_attest_evidence(dev_id, &ctx, evidence, &evidence_len);
if (ret != 0){
   //记录日志
   return ret;
}
…
```
