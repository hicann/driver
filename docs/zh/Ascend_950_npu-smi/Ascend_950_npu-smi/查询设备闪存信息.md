## 使用示例<a name="zh-cn_topic_0000001220096534_zh-cn_topic_0000001117597256_section66051714132117"></a>

查询设备0的闪存信息。

```bash
npu-smi info -t flash -i 0
```

```text
        NPU ID                         : 0
        Flash Count                    : 1

        Flash ID                       : 1730504
        Manufacturer ID                : 0xC2
        Capacity(MB)                   : 64
```
