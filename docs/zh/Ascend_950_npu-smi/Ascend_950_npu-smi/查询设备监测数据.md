# 查询设备监测数据

## 功能说明

`npu-smi info watch`用于查询所有设备的监测数据。

`npu-smi info watch -i dev_id`用于查询指定设备的监测数据。

`npu-smi info watch -i dev_id -d delay_seconds -s watch_type`用于查询指定设备和监测类型的监测数据。

## 命令格式

```bash
npu-smi info watch
npu-smi info watch -i dev_id
npu-smi info watch -i dev_id -d delay_seconds -s watch_type
```
