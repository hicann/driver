# 设置结束日志进程并清除配置<a name="ZH-CN_TOPIC_0000002583332666"></a>

## 功能说明<a name="section143092264417"></a>

用于结束所有持续收集设备日志的进程，并清除持续收集所有设备日志的相关配置。

## 命令格式<a name="section13508430152616"></a>

`npu-smi set -t clear-syslog-cfg`

## 参数说明<a name="section1518211201813"></a>

无
