## 功能说明

## 命令格式

`npu-smi upgrade -t item -i dev_id -f file_path`

## 使用指南<a name="zh-cn_topic_0000001264656709_zh-cn_topic_0000001117597252_section1882494910505"></a>

- 该命令不支持并发升级VRD固件。
- VRD固件升级过程中，请勿重复执行升级VRD操作，否则会返回升级失败，但不会影响当前VRD固件升级操作执行。
- VRD固件升级后需要执行**reboot**重启系统进行生效，等待至少60秒后再进行DC下电再上电操作，等待NPU成功建链。

## 使用示例<a name="zh-cn_topic_0000001264656709_zh-cn_topic_0000001117597252_section66051714132117"></a>

升级设备0中的VRD固件。

```bash
npu-smi upgrade -t vrd -i 0 -f Atlas-A5-hdk-vrd-firmware_xxx.hpm
```

```text
 [WARNING]: Do not power off or restart the system during the upgrade.     
        Transfer                      : success
        Status                        : start to upgrade
Start upgrade [100].                   
        Message                       : Upgrade device success
```

> **说明：** 
> 
> Atlas-A5-hdk-vrd-firmware_\_xxx._hpm请用实际的固件包名替换。
