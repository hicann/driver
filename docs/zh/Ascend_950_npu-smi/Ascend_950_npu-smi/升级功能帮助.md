## 使用示例<a name="zh-cn_topic_0000001459123601_zh-cn_topic_0000001457338873_zh-cn_topic_0000001264536705_zh-cn_topic_0000001117597254_section66051714132117"></a>

显示升级命令的帮助信息。

```bash
npu-smi upgrade -h
```

```text
Usage: npu-smi upgrade <-h|-t type|-b type> [Options...]

Commands:
       -h, --help     Show this help text and exit
       -t type        Upgrade device firmware
                      type: vrd
       -b type        Query the firmware version
                      type: vrd vdm

Options:
       -i ID          Upgrade for this device ID
       -f file        The firmware file path
```
