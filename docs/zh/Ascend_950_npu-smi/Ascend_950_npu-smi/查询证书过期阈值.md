# 查询证书过期阈值

## 功能说明<a name="zh-cn_topic_0000001937698753_section32081917543"></a>

用于查询设置的TLS过期告警阈值。

## 命令格式<a name="zh-cn_topic_0000001937698753_section1120941185416"></a>

`npu-smi info -t tls-cert-period -i dev_id`
