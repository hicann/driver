# 查询设备映射关系信息

## 功能说明

## 命令格式<a name="zh-cn_topic_0000001264416741_zh-cn_topic_0000001163757109_section11902434155614"></a>

`npu-smi info -m`

## 参数说明<a name="zh-cn_topic_0000001264416741_zh-cn_topic_0000001163757109_section52928161114"></a>

无
