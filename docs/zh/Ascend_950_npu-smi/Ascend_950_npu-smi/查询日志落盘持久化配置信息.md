# 查询日志落盘持久化配置信息<a name="ZH-CN_TOPIC_0000002613572517"></a>

## 功能说明<a name="section394911416273"></a>

用于查询设备日志落盘持久化配置信息，包括日志落盘配置持久化开关模式、日志容量档位、日志存放路径和持续收集设备日志的进程。

## 命令格式<a name="section49511814152711"></a>

`npu-smi info -t sys-log-info`

## 参数说明<a name="section1518211201813"></a>

无

## 使用指南<a name="section796341422718"></a>

如果日志落盘持久化配置信息查询失败，返回“ERROR: The configuration file has been maliciously modified, please clear configuration and stop syslog collecting process first. Failed to query "sys-log-info" info.”，则用户需要先执行**npu-smi set -t clear-syslog-cfg**命令将配置文件恢复为默认配置且结束日志落盘相关进程，再查询日志落盘持久化配置信息。

