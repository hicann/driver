# 查询固件版本信息<a name="ZH-CN_TOPIC_0000002613692377"></a>

## 功能说明<a name="zh-cn_topic_0000001220416518_zh-cn_topic_0000001163797115_section18674262006"></a>

用于查询固件的版本信息。

## 命令格式<a name="zh-cn_topic_0000001220416518_zh-cn_topic_0000001163797115_section386902615016"></a>

`npu-smi upgrade -b item -i dev_id`

## 使用示例<a name="zh-cn_topic_0000001220416518_zh-cn_topic_0000001163797115_section66051714132117"></a>

```bash
npu-smi upgrade -b vrd -i 0
```
