# 查询基本信息<a name="ZH-CN_TOPIC_0000002583332600"></a>

## 功能说明<a name="zh-cn_topic_0000001264656717_zh-cn_topic_0000001117597270_section10160164417116"></a>

用于查询所有设备的基本信息。

## 命令格式<a name="zh-cn_topic_0000001264656717_zh-cn_topic_0000001117597270_section1516414419110"></a>

`npu-smi info`

## 参数说明<a name="zh-cn_topic_0000001264656717_zh-cn_topic_0000001117597270_section52928161114"></a>

无

## 使用指南<a name="zh-cn_topic_0000001264656717_zh-cn_topic_0000001117597270_section2106140121613"></a>

- 在空闲状态，由于TaiShan核启动加载，会有35M\~45M的片上内存占用，属于正常现象。
- 当设备上未运行进程时，回显信息中会显示“No running processes found in NPU  _dev\_id_”。
- 每个芯片上最多显示8个device侧SVM相关的进程信息。
- 在宿主机和特权容器场景下执行该命令时可查询到宿主机、普通容器、特权容器中运行的所有进程，在普通容器场景下执行该命令时仅能查询到普通容器中运行的所有进程。

## 使用示例<a name="zh-cn_topic_0000001264656717_zh-cn_topic_0000001117597270_section15164444410"></a>

查询所有设备的基本信息。

```bash
 npu-smi info
```
