# 查询设备超节点信息<a name="ZH-CN_TOPIC_0000002583172620"></a>

## 功能说明<a name="section1597791992519"></a>

用于查询设备的超节点信息。

## 命令格式<a name="section1751092614258"></a>

`npu-smi info -t spod-info -i dev_id`

## 使用示例<a name="section738295551114"></a>

查询设备0的超节点信息。

```bash
npu-smi info -t spod-info -i 0
```

```text
This device does not support querying spod-info.
```

注：因为本设备不支持此命令，所以回显为“This device does not support  _xxx_”。
