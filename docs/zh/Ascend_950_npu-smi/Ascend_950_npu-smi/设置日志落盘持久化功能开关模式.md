# 设置日志落盘持久化功能开关模式<a name="ZH-CN_TOPIC_0000002613572461"></a>

## 功能说明<a name="section689116327160"></a>

用于设置日志落盘持久化功能开关模式。

## 命令格式<a name="section2965174417182"></a>

`npu-smi set -t sys-log-enable -d mode`

## 使用指南<a name="section796341422718"></a>

- 该命令为持久化命令，重启系统后日志落盘持久化功能开关模式不变。
- 如果日志落盘配置持久化开关模式设置失败，返回“ERROR: The configuration file has been maliciously modified, please clear configuration and stop syslog collecting process first.”，则用户需要先执行**npu-smi set -t clear-syslog-cfg**命令将配置文件恢复为默认配置且结束日志落盘相关进程，再设置日志落盘持久化功能开关模式。

## 使用示例<a name="section825263010207"></a>

设置日志落盘为持久化功能开启模式。

```bash
npu-smi set -t sys-log-enable -d 1
```

```text

```
