## 使用示例<a name="zh-cn_topic_0000001264416749_zh-cn_topic_0000001117437340_section66051714132117"></a>

查询设备0的健康状态。

```bash
npu-smi info -t health -i 0
```

```text
        NPU ID                         : 0
        Health                         : OK
        Error Code                     : NA
        Error Information              : NA
```
