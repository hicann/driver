## 命令功能

查询msnpureport工具版本信息。

## 注意事项

无。

## 命令格式

```sh
msnpureport version
```

## 参数说明

无

## 使用示例

1. 登录Host侧服务器。
2. 执行以下命令。

   ```sh
   msnpureport version
   ```
